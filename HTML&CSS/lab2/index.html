HTTP

HTTP (Hypertext Markaup Language - Giao thức truyền tải siêu văn bản) là giao thức giao tiếp giữa chương trình cung cấp dịch vụ WWW(wb server) và chương trình sử dụng www (Trình duyệt web). Đây là giao thức ở tầng ứng dụng của dịch vụ của boo giao thức TCP/IP, giao thức nền tảng của Internet

Phiên bản mới nhất của bộ giao thức này là HTTP 1.1. Phiên bản này bổ sung một số đặc tả, lệnh và một số quy định mới nhằm nâng cao khả năng giao tiếp giữa trình duyệt web và web server
